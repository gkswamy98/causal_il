# CausIL
A repo for Causal Imitation Learning.

## Running Experiments
To re-train the LunarLander expert, run:
```bash
python experts/train_expert.py
```
To train a learner, run:
```bash
jupyter notebook
```
and open up CausIL.ipynb.

## Visualizing Results
Run:
```bash
jupyter notebook
```
and open up vis.ipynb.
